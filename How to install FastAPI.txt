
Crear entorno virtual Python
python3 -m venv nuevo_entorno
source nuevo_entorno/bin/activate


Instalar FastAPI desde Entorno
pip install fastapi uvicorn


Como usar FastAPI
cd VigilancIA_FastAPI
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
